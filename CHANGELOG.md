# Changelog

All notable changes to GROVE will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

## [v1.2.0] - 2026-03-23

### Added
- One-command installer for all platforms
- One-click installer and distribution package
- Dry-run mode for safe testing
- Templates for consistent project setup
- Task prioritizer for workflow optimization
- Original idea tracking and comparison feature

### Fixed
- Resolved all issues from technical review

## [v1.1.0] - 2026-03-20

### Added
- GROVE Spec v2 - Complete rewrite with deep decomposition
- SDD integration to Spec and Opti
- Engram integration for persistent memory

## [v1.0.0] - 2026-03-15

### Added
- Initial release of GROVE ecosystem
- SDD integration
- Engram client
- Dashboard
- Edge case tests
